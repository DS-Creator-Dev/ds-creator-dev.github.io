Get Started

Introduction

Using DS Creator, you can make your own DS games. Developed especially for beginners, DS Creator makes it easy to learn how to program and create your own games, without needing to know complicated coding languages.

Installation

Installing DS Creator is simple. Simply download DS Creator and install devkitPro's libnds. Now you are ready to make DS games!

Creating DS Games

In DS Creator, you use scenes, objects, and events to create your dream game!
